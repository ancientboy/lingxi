## 🏗️ 技术架构

### 项目结构

```
android-app/
├── app/
│   ├── src/
│   │   ├── main/
│   │   │   ├── java/com/lingxi/app/
│   │   │   │   ├── MainActivity.kt          # 主 Activity
│   │   │   │   ├── WebViewClient.kt         # WebView 客户端
│   │   │   │   ├── WebAppInterface.kt       # JS 桥接
│   │   │   │   ├── VoiceRecognition.kt      # 语音识别
│   │   │   │   ├── FileChooser.kt           # 文件选择
│   │   │   │   └── SplashScreen.kt          # 启动页
│   │   │   ├── res/
│   │   │   │   ├── layout/
│   │   │   │   │   └── activity_main.xml    # 主布局
│   │   │   │   ├── values/
│   │   │   │   │   ├── colors.xml           # 颜色
│   │   │   │   │   ├── strings.xml          # 字符串
│   │   │   │   │   └── styles.xml           # 样式
│   │   │   │   ├── drawable/
│   │   │   │   │   └── ic_launcher.xml      # 图标
│   │   │   │   └── mipmap/
│   │   │   │       └── ic_launcher.png      # 启动图标
│   │   │   └── AndroidManifest.xml          # 清单文件
│   │   └── test/                            # 单元测试
│   └── build.gradle                         # 构建配置
├── gradle/                                  # Gradle wrapper
├── build.gradle                             # 项目构建
├── settings.gradle                          # 项目设置
└── README.md                                # 说明文档
```

### 技术栈

- **语言：** Kotlin 1.9.0
- **最低 SDK：** Android 5.0 (API 21)
- **目标 SDK：** Android 14 (API 34)
- **构建工具：** Gradle 8.0
- **WebView：** Android WebView
- **语音识别：** Android SpeechRecognizer
- **文件选择：** Android FileChooser
