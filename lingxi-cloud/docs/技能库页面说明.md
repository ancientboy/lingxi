## 🎯 页面功能

- ✅ 顶部搜索 + 筛选功能
- ✅ 响应式设计（移动端适配）

**访问方式**:
- 直接访问：`/skills.html`
- 从 chat.html 导航栏点击"📚 技能库"

## 🔧 API 接口

### 获取技能库
```
GET /api/skills/library
Headers: Authorization: Bearer {token}
```

### 获取已安装技能
```
GET /api/skills/installed
Headers: Authorization: Bearer {token}
```

### 安装技能
```
POST /api/skills/install/{skillId}
Headers: Authorization: Bearer {token}
```

## 🚀 使用流程

1. **访问技能库**
   - 点击导航栏"📚 技能库"
   - 或直接访问 `/skills.html`

2. **浏览技能**
   - 左侧选择 Agent 分类
   - 右侧查看技能卡片

3. **搜索/筛选**
   - 顶部搜索框输入关键词
   - 点击筛选按钮（全部/已安装/最新）

4. **安装技能**
   - 点击"安装"按钮
   - 等待安装完成
   - 按钮变为"✓ 已安装"

## 📱 响应式断点

- **桌面端**: > 768px
- **移动端**: ≤ 768px

移动端特性：
- Agent 列表变为网格布局
- 技能卡片单列显示
- 导航链接隐藏（简化）
