## 二、收费模式（混合制）

### 免费用户
- 每日 100 积分（每天重置）
- 约 10-20 次对话
- 引导升级到付费

### 订阅用户（月付）
| 套餐 | 价格 | 积分/月 | 特权 |
|------|------|---------|------|
| 基础版 | ¥29/月 | 5,000 | - |
| 专业版 | ¥99/月 | 20,000 | - |
| 企业版 | ¥299/月 | 100,000 | 优先支持 |

### 按量付费
- ¥10 = 1,000 积分
- 积分永不过期

## 三、积分消耗规则

| 模型 | 输入积分/1K tokens | 输出积分/1K tokens |
|------|-------------------|-------------------|
| qwen-plus | 1 | 2 |
| qwen-max | 2 | 4 |
| glm-4 | 1.5 | 3 |

**示例**：一次对话约 500 输入 + 200 输出 = 0.9 积分

## 六、API 接口

### 获取使用量统计
```
GET /api/user/usage

Response: {
  "success": true,
  "data": {
    "totalTokens": 1250000,
    "totalRequests": 3520,
    "today": { "tokens": 3200, "requests": 12 },
    "week": { "tokens": 25000, "requests": 85 },
    "month": { "tokens": 125000, "requests": 420 },
    "byModel": { ... },
    "recentUsage": [
      { "date": "2026-03-02", "tokens": 3200, "requests": 12 },
      ...
    ]
  }
}
```

### 消耗积分（内部）
```
POST /api/credits/deduct (内部调用)

Body: {
  "userId": "xxx",
  "model": "qwen-plus",
  "inputTokens": 500,
  "outputTokens": 200
}

Response: {
  "success": true,
  "deducted": 0.9,
  "newBalance": 3499.1
}
```
