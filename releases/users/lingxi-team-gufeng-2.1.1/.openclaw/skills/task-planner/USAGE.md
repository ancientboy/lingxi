## 使用方式

### 自然语言（推荐）

直接说想法，灵犀自动识别并派发：

```
我想学习 [技能]
帮我规划 [目标]
我需要制定 [计划]
```

**示例：**
- "我想学习 React"
- "帮我规划健身计划"
- "我需要制定阅读计划"
- "想学习英语，3 个月达到流利"

### 手动调用

```javascript
import planner from './skills/task-planner/index.mjs';

// 创建计划
const plan = planner.createPlan('学习 AI', {
  deadline: '2026-06-01',
  hoursPerWeek: 10,
  tags: ['学习']
});

// 手动添加阶段
planner.addStage(plan.id, {
  name: '基础准备',
  duration: '2 weeks'
});

// 添加任务
planner.addTaskToStage(plan.id, stageId, {
  title: 'Python 基础',
  hours: 5,
  priority: 'high'
});

// 查看进度
const progress = planner.trackProgress(plan.id);
```

## 工作流程

```
1. 用户提出目标
   ↓
2. 若曦分析目标
   - 评估难度
   - 估算总时长
   - 拆解成阶段
   ↓
3. 创建计划
   - 生成阶段
   - 创建任务
   - 估算时长
   ↓
4. 智能安排
   - 根据可用时间
   - 安排到日程
   - 同步到 tasks
   ↓
5. 进度追踪
   - 完成任务
   - 更新进度
   - 自动调整
```

## 若曦的角色

**若曦（ops）负责：**

1. **分析目标**
   - 评估可行性
   - 估算时间
   - 识别关键里程碑

2. **拆解任务**
   - 把大目标拆成小任务
   - 设定优先级
   - 估算每个任务时长

3. **智能安排**
   - 根据用户时间安排
   - 避免冲突
   - 考虑优先级

4. **追踪进度**
   - 显示进度条
   - 计算预计完成时间
   - 自动调整计划

**若曦的说话风格：**

```
"数据会告诉我们答案。根据你的时间投入，我建议这样安排..."
"从进度来看，你完成得很好！继续保持这个节奏..."
"我注意到你最近进度有点慢，需要调整计划吗？"
```
